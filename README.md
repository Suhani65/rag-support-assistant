🚀 Built a RAG-Based Customer Support Assistant using LangGraph + LLMs

I recently completed an internship project where I designed and implemented a Retrieval-Augmented Generation (RAG) system from scratch.

🔍 Key Features:

• PDF knowledge base ingestion
• Semantic search using embeddings (ChromaDB)
• Context-aware response generation (FLAN-T5)
• Graph-based workflow using LangGraph
• Intent detection & conditional routing
• Human-in-the-Loop (HITL) escalation

🧠 This project helped me understand how real-world AI systems are designed beyond just chatbots — focusing on architecture, workflows, and decision-making.

📌 Tech Stack:
Python | LangChain | LangGraph | HuggingFace | ChromaDB

▶️ How to Run:

1. Clone the repo
2. Install dependencies (`pip install -r requirements.txt`)
3. Run: `python rag_pipeline.py`
4. Ask your query in the terminal
